# 创作区 v8 — 视觉需求稿

> 版本：v8.1（精简规范版）
> 配套 mockup：`docs/mockups/creation-zone-v8.html`
> 设计 Token 来源：`app/globals.css`、`docs/visual.md`
> 更新日期：2026-07-12

---

## 一、设计原则

| 原则 | 说明 |
|------|------|
| 章纲是核心 | 章纲占主工作区，正文作为 Tab 切换查看，AI 面板为独立生成态 |
| 蓝图是参考 | 总纲/卷纲始终以一行参考条形式可见，不可偏离 |
| 视觉极简 | 所有设计严格使用 globals.css Token，无硬编码颜色、无装饰性元素 |
| 信息层级分明 | 通过 `text-primary/secondary/tertiary/light` 四级文字色区分信息权重 |

## 三、Topbar（顶部导航栏）

### 视觉规格

| 属性 | 值 |
|------|-----|
| 高度 | 56px（`--topbar-height`） |
| 背景 | `--bg-elevated` |
| 底边框 | 1px solid `--border` |
| 左侧内容 | 品牌名 + 分割线 + 书名 |
| 右侧内容 | 搜索按钮 + 设置按钮 |

### 元素明细

| 元素 | 字体 | 字号 | 颜色 | 说明 |
|------|------|------|------|------|
| 品牌名 "AI Writer" | `--font-display` | 16px | `--text-primary` | serif 字体，`font-weight:600`，`letter-spacing:.03em` |
| 分割线 | — | 1px 宽 | `--border` | 高度 16px，垂直居中 |
| 书名 "夜行" | `--font-body` | 13px | `--text-tertiary` | 点击可跳转书库 |
| 搜索/设置按钮 | SVG 图标 | 14px | `--text-tertiary` | 32×32px 点击区域，`border-radius: var(--radius-md)` |

### 交互

| 操作 | 行为 |
|------|------|
| 悬停按钮 | 背景变 `--bg-muted`，图标色变 `--text-primary` |
| 点击书名 | 跳转到书库页面（路由 `/books`） |

## 四、Activity Bar（活动栏）

### 视觉规格

| 属性 | 值 |
|------|-----|
| 宽度 | 42px |
| 背景 | `--bg-strong` |
| 右边框 | 1px solid `--border` |
| 图标风格 | SVG 线条图标，16×16px，`opacity:.6`（默认） |
| 间距 | 按钮之间 2px gap |

### 图标列表（从上到下）

| 图标 | 标题 | 说明 |
|------|------|------|
| 书籍信息 | 图书图标 | 矩形 + 横线 |
| 世界规则 | 地球图标 | 圆形 + 椭圆 + 横线 |
| 设定库 | 列表图标 | 三条横线矩形 |
| 标签库 | 标签图标 | 菱形标签 |
| **创作区** | **编辑图标** | **当前激活** |
| 伏笔库 | 图钉图标 | 圆形 + 向下箭头 |
| 正文库 | 网格图标 | 四宫格矩形 |
| 事实库 | 文档图标 | 矩形 + 横线 |
| 收起 | 左箭头 | 底部，分隔 |

### 交互

| 状态 | 视觉表现 |
|------|----------|
| 默认 | 图标色 `--text-light` |
| 悬停 | 背景 `--bg-muted`，图标色 `--text-secondary` |
| 激活 | 背景 `--color-primary-bg`，图标色 `--color-primary` |
| 激活 indicator | 左侧 2px 竖线，颜色 `--color-primary`，位置 `top:8px bottom:8px` |

### 点击行为

| 按钮 | 行为 |
|------|------|
| 切换面板 | 替换 Nav 区域内容（导航树/设定库/伏笔库等） |
| 当前面板已激活 | 无操作 |
| 底部收起按钮 | 折叠 Nav 区域，Center 自动扩展 |

## 五、Nav（导航树 / 大纲目录）

### 视觉规格

| 属性 | 值 |
|------|-----|
| 宽度 | 210px |
| 背景 | `--bg-elevated` |
| 右边框 | 1px solid `--border-light` |
| 内边距 | 6px |

### 头部

| 属性 | 值 |
|------|-----|
| 内边距 | 10px 14px |
| 底边框 | 1px solid `--border-light` |
| 标题 | "大纲"，11px，`--text-tertiary`，`letter-spacing:.04em`，`text-transform:uppercase` |
| 新建按钮 | "+" 符号，22×22px 点击区 |

### 导航项 — 总纲

| 属性 | 值 |
|------|-----|
| 层级 | 顶层 |
| 样式 | 同普通导航项（见下方），点击进入总纲编辑视图 |

### 导航项 — 卷组（Volume Group）

| 属性 | 值 |
|------|-----|
| 折叠头 | 左箭头 + 卷标题 + 章数 badge |
| 卷标题 | 12px，`--text-secondary`，如 "第一卷 · 迷雾觉醒" |
| badge | 9px，`padding:1px 5px`，`--border-radius-full`，背景 `--bg-muted`，文字 `--text-light` |
| 箭头 | 8px 三角形，旋转动画 `.2s` |

#### 折叠/展开交互

| 状态 | 行为 |
|------|------|
| 展开 | 箭头向下（默认），子章列表可见 |
| 收起 | 箭头旋转 -90°（CSS class `.col`），子章列表 `display:none` |
| 点击折叠头 | 切换展开/收起状态 |
| 展开动画 | 无过渡动画，直接切换 display（保持简洁） |

### 导航项 — 章节（Chapter Item）

| 属性 | 值 |
|------|-----|
| 缩进 | 左侧 padding 16px |
| 序号 | 10px，`--text-light`，等宽字体，宽度 16px 右对齐 |
| 标题 | 12px，`--text-secondary` |
| 点击区 | 32px 高度行（padding 4px 10px） |

#### 选中状态

| 属性 | 值 |
|------|-----|
| 文字色 | `--color-primary` |
| 字重 | 500 |
| 背景 | `--color-primary-bg` |
| 左侧 indicator | 2px 竖线，`--color-primary`，位置 `top:6px bottom:6px` |

### 交互总结

| 操作 | 行为 |
|------|------|
| 点击总纲 | 右侧 Center 切换到总纲编辑视图（全屏表单） |
| 点击卷标题 | 展开/收起该卷的章节列表 |
| 点击章节 | 1. 右侧 Center 切换到该章的章纲/正文 Tab 视图；2. 导航项高亮 |
| 新建按钮 "+" | 弹出选择：新建卷 / 在当前卷下新建章 |
| hover 导航项 | 背景变 `--bg-muted` |
