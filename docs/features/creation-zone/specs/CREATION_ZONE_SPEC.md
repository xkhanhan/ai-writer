# 创作区设计文档

## 概述

将现有书籍工作区的"内容区域"active 改造为"创作区"，并新增"正文库"active。采用滚动规划型创作流程，支持用户手动创建总纲、卷纲、章纲、正文。后续将接入 AI 辅助生成（公用 AI 组件 + 工作流引擎）。

## 设计决策

### 滚动规划型（四层结构）

| 层次 | 清晰度 | 规划范围 | 说明 |
|------|--------|---------|------|
| 总纲 | 模糊 | 整本书方向 | 只有方向，不细化细节，随写随调 |
| 卷纲 | 详细 | 只规划当前这一卷 | 核心冲突、发展弧线、重要节点 |
| 章纲 | 滚动 | 只规划接下来 3-5 章 | 写完几章，再规划下一批 |
| 正文 | 实时 | 基于章纲写作 | 写作可反哺调整上层规划 |

### Active 结构

| Active | 职责 | 状态 |
|--------|------|------|
| 书籍信息 | 书名、简介、世界观、人设等 | 保留 |
| 世界规则 | 世界观设定 | 保留 |
| 设定库 | 角色/势力设定 | 保留 |
| 创作区 | 原"内容区域"改造，手动 + AI 创作工作区 | 改造 |
| 正文库 | 存稿备份，独立于创作区 | 新增 |

### 创作区与正文库的关系

- 创作区正文 = 正在编辑中的内容
- 正文库 = 点击"存稿"后备份的版本（可覆盖、可删除）
- 两者独立，互不影响

## 创作区设计

### 整体布局：两栏结构

```
┌──────────────┬─────────────────────────┐
│  左侧导航栏   │     右侧内容区           │
│  (280px)     │     (flex: 1)           │
│              │                         │
│  - 总纲      │  根据点击展示不同内容：   │
│  - 第一卷    │  - 卷纲内容卡片          │
│    - 第1章   │  - 卷纲编辑表单          │
│    - 第2章   │  - 章纲内容卡片          │
│    - 第3章   │  - 章纲编辑表单          │
│  - 第二卷    │  - 正文编辑器            │
│  + 新建卷    │  - 空状态引导            │
└──────────────┴─────────────────────────┘
```

### 左侧导航栏

- 顶部：标题"创作区"
- 总纲入口（固定，带朱砂红左边框）
- 卷列表（可折叠树形结构）：
  - 卷标题行：展开/折叠箭头 + 卷名 + 章节进度
  - 展开后：章纲列表，每项显示序号、标题、状态标识
  - 章纲状态颜色：翠绿（已写）、金色（撰写中）、灰色（已规划）
- 底部：新建卷按钮

### 右侧内容区（8 种视图）

#### 视图 1：空状态引导

新书初次进入，没有任何卷。右侧展示引导：
- 标题"开始你的创作"
- 建议创作流程说明
- 两个按钮：编辑总纲 / 新建第一卷

#### 视图 2：点击卷标题 - 卷纲内容卡片

展示卷纲信息（只读）：
- 核心冲突
- 发展弧线（流程图式：节点 → 节点）
- 重要节点（章节位置 + 描述）
- 预计看点（标签形式）
- 右上角按钮：编辑卷纲

#### 视图 3：点击章纲 - 章纲内容卡片

展示章纲信息（只读），字段丰富：
- 情节概要
- 场景设定（标签）
- 出场人物（标签）
- 重要情节（有序列表）
- 伏笔设置（列表）
- 预计看点（标签）
- 预计字数 / 当前字数
- 右上角按钮：编辑章纲 / 写正文

#### 视图 4：编辑卷纲表单

右侧切换为表单，字段：
- 卷标题 *（必填）
- 核心冲突（textarea）
- 发展弧线（textarea）
- 重要节点（动态列表：章节位置 + 描述 + 删除按钮 + 添加按钮）
- 预计看点（逗号分隔输入）
- 底部：取消 / 保存

#### 视图 5：编辑章纲表单

右侧切换为表单，字段：
- 序号 *（必填）
- 章节标题 *（必填）
- 情节概要 *（textarea）
- 场景设定（标签输入器：回车添加，× 删除）
- 出场人物（标签输入器）
- 重要情节（有序动态列表）
- 伏笔设置（动态列表）
- 预计看点（逗号分隔）
- 预计字数
- 备注
- 底部：取消 / 保存 / 保存并写正文

#### 视图 6：正文编辑器

- 顶部：章节标题 + 状态标签 + 按钮（返回章纲 / 存稿）
- 章纲参考条（可折叠）：显示当前章纲概要
- 正文编辑区（书法字体，line-height 2）
- 底部状态栏：字数 + 自动保存 + 上一章/下一章

#### 视图 7：总纲编辑

- 总纲编辑（带提示："总纲保持模糊，随写作推进逐步细化"）
- 字段：
  - 整体方向（textarea）
  - 阶段划分（textarea，每行一个阶段）
  - 核心卖点（逗号分隔）

## 正文库设计

### 布局：两栏结构

左侧：存稿列表（章节名 + 字数 + 日期 + 删除按钮）
右侧：正文预览（只读）

### 特性

- 存稿来源：创作区点击"存稿 →"按钮，覆盖对应章节
- 可删除存稿
- 独立于创作区，删除正文库不影响创作区内容

## 数据结构

### 总纲

```typescript
interface BookOutline {
  direction: string;       // 整体方向
  stages: string;          // 阶段划分
  sellingPoints: string;   // 核心卖点
}
```

### 卷纲

```typescript
interface VolumeOutline {
  id: string;
  title: string;           // 卷标题
  coreConflict: string;    // 核心冲突
  developmentArc: string;  // 发展弧线
  keyPoints: Array<{       // 重要节点
    chapter: string;       // 约X章
    description: string;   // 节点描述
  }>;
  highlights: string;      // 预计看点
  order: number;           // 卷序号
}
```

### 章纲

```typescript
interface ChapterOutline {
  id: string;
  volumeId: string;        // 所属卷
  order: number;           // 章节序号
  title: string;           // 章节标题
  summary: string;         // 情节概要
  scenes: string[];        // 场景设定
  characters: string[];    // 出场人物
  keyEvents: string[];     // 重要情节
  foreshadowings: string[];// 伏笔设置
  highlights: string;      // 预计看点
  expectedWords: number;   // 预计字数
  currentWords: number;    // 当前字数
  status: 'planned' | 'writing' | 'done'; // 状态
  content?: string;        // 正文内容
  note?: string;           // 备注
}
```

### 正文库存稿

```typescript
interface ArchivedChapter {
  id: string;
  chapterId: string;       // 对应章纲 ID
  order: number;
  title: string;
  content: string;
  wordCount: number;
  archivedAt: string;      // 存稿时间
}
```

## 视觉规范

遵循项目东方水墨风格（`app/globals.css`）：

## 后续扩展（AI 辅助）

### AI 生成公用组件

所有需要 AI 的地方共用一套组件，流程：
1. 用户输入区（可定制字段）
2. 提示词预览区（程序自动组装，用户可查看编辑）
3. 确认按钮
4. AI 生成中（Loading）
5. 结果展示区（用户审核、可修改、可重新生成）
6. 确认通过 → 进入下一步

### 工作流引擎

支持多节点 AI 调用：
- AI 节点：调用模型，有独立提示词
- 程序节点：查找、组装等非 AI 逻辑
- 节点输出 = 下一节点输入
- 初期内置固定工作流，后期支持用户自定义

### AI 接入点

| 业务 | 输入 | 输出 |
|------|------|------|
| 总纲生成 | 整体方向 | 模糊总纲 |
| 卷纲生成 | 书籍信息 + 用户想法 + 补充 | 详细卷纲 |
| 章纲滚动规划 | 已写内容 + 当前卷纲 | 下 3-5 章章纲 |
| 正文生成 | 章纲 + 设定库信息 + 工作流 | 正文内容 |
