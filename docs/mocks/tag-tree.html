TagTree 视觉稿件

基于 antd v6 Tree 组件 + 项目视觉规范的标签树设计。本方案遵循视觉规范，不使用 :global(.ant-xxx) 覆盖 antd 样式，改用 ConfigProvider tokens + CSS 变量实现自定义。

A 完整树结构 多层级 + 展开态 + 操作按钮
设计思路：
  • 使用 antd Tree 的 blockNode + showLine={{ showLeafIcon: false }}
  • 节点高度由 ConfigProvider token controlHeightSM: 36 控制
  • 缩进由 token indentSize: 24 控制
  • 节点内容（名称 + 描述 + 操作按钮）在 title prop 中自定义渲染
  • 选中/悬停高亮由 antd 内置机制处理，通过 ConfigProvider token 注入项目色彩

D 实现方案 ConfigProvider tokens + CSS Modules
合规要点（视觉规范 §1）：
  • 禁止 使用 :global(.ant-xxx) 覆盖 antd 样式
  • 推荐 使用 ConfigProvider theme.components.Tree 注入 token
  • 允许 在 CSS Modules 中编写自定义布局样式（仅作用于 title prop 中的自定义 DOM）
ConfigProvider 配置
// 在标签库组件中包裹 ConfigProvider <ConfigProvider theme={{ components: {{ Tree: { // 节点高度 controlHeightSM: 36, // 缩进宽度 indentSize: 24, // 悬停背景色 — 对齐项目 --bg-muted colorBgContainer: '#faf8f4', // 选中背景色 — 对齐项目 --color-primary-bg colorBgTreeNodeHover: 'rgba(47,93,80,0.05)', colorBgTreeNodeSelected: 'rgba(47,93,80,0.05)', // 展开箭头颜色 colorTreeExpandIconToggle: '#807b74', // 连接线颜色 colorLine: '#ebe7e0', } } }} > <TagTree ... /> </ConfigProvider>
CSS Modules（仅 title 内容）
/* tag-tree/index.module.css */ /* 注意：不使用 :global(.ant-xxx)，所有样式仅作用于 title 中的自定义 DOM */ /* 节点内容包装 — 作为 title prop 的根元素 */ .nodeWrap { display: flex; align-items: center; gap: 8px; width: 100%; min-width: 0; padding: 6px 0; } /* 名称 + 描述容器 */ .nodeText { flex: 1; min-width: 0; display: flex; flex-direction: column; gap: 1px; overflow: hidden; } /* 名称 */ .name { font-size: 13px; font-weight: 500; color: var(--text-primary); line-height: 1.4; overflow: hidden; text-overflow: ellipsis; white-space: nowrap; } /* 描述 */ .desc { font-size: 12px; color: var(--text-tertiary); line-height: 1.4; overflow: hidden; text-overflow: ellipsis; white-space: nowrap; } /* 操作按钮区 */ .actions { flex-shrink: 0; display: flex; align-items: center; gap: 2px; opacity: 0; transition: opacity 0.15s; } .nodeWrap:hover > .actions { opacity: 1; } .actionBtn { width: 26px; height: 26px; display: inline-flex; align-items: center; justify-content: center; border-radius: var(--radius-sm); color: var(--text-tertiary); cursor: pointer; transition: all 0.15s; font-size: 13px; } .actionBtn:hover { background: var(--color-primary-bg); color: var(--color-primary); } /* 搜索高亮 */ .highlight { background: var(--color-primary-bg); color: var(--color-primary); padding: 0 4px; border-radius: 3px; }

F v1 → v2 变更对比 改进点说明
v1 问题（当前代码）
问题 1 使用 :global(.ant-tree-treenode) 等选择器覆盖 antd 样式 → 违反视觉规范 §1
问题 2 节点 padding 4px 8px → 与 antd 默认行高不匹配
问题 3 选中态用 !important 强制覆盖 → 不优雅且可能被主题覆盖
v2 改进
改进 1 移除所有 :global(.ant-xxx) 选择器，改用 ConfigProvider tokens
改进 2 CSS Modules 仅作用于 title prop 中的自定义 DOM（.nodeWrap / .actions 等）
改进 3 antd 内置选中/悬停高亮通过 token 注入项目色彩，无需 !important
