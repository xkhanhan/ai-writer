DAiPanel 纯黑盒组件总览
设计原则
面板知道的
- 当前在哪个面板/场景
- 要生成哪些字段
- 有哪些提示词模板
- 用户输入了什么想法
- 用户编辑/采纳了哪些卡片
面板不知道的
- 字段保护/锁定逻辑
- 采纳后数据写入哪里
- 表单的校验规则
- 业务规则
- 数据持久化
组件接口
<AiPanel
  context={{ panel, scenario }}
  fields={[{ key, label }]}
  programData={{...}}
  promptTemplates={[{ id, name }]}
  defaultTemplateId="xxx"
  onGenerate={({ input, templateId }) => ...}
  onAdopt={(results) => ...}
/>
复用场景
场景panelscenariofields模板
总纲outlineoutline_generate故事核心/冲突/结局总纲提示词库
卷纲volumevolume_generate卷主题/卷冲突卷纲提示词库
角色charactercharacter_create性格/背景/动机角色提示词库
书籍信息bookbook_info题材/文风/受众书籍提示词库
数据流
调用层传入 → AiPanel 渲染 → onGenerate → 调 AI → AiPanel 渲染卡片 → onAdopt → 调用层处理
完全单向。面板不主动写入数据，不保存状态，不做业务判断。纯展示+交互的黑盒。
