# AI Agent 面板设计稿

## 1. 设计概述

### 1.1 设计原则
- **一致性**：与系统现有的暖纸色系风格保持一致
- **层次感**：通过背景色和边框区分不同区域
- **简洁性**：避免过度装饰，突出核心功能
- **可操作性**：工具调用结果支持一键操作

### 1.2 设计规范
| 属性 | 值 |
|------|-----|
| 主色 | `var(--color-primary)` #2F5D50 |
| 背景色 | `var(--bg-page)` / `var(--bg-elevated)` |
| 边框色 | `var(--border)` / `var(--border-light)` |
| 文字色 | `var(--text-primary)` / `var(--text-secondary)` / `var(--text-tertiary)` |
| 圆角 | `var(--radius-sm)` / `var(--radius-md)` |
| 间距 | `var(--space-2)` / `var(--space-3)` / `var(--space-4)` |
| 字体 | `var(--font-body)` / `var(--font-mono)` |

## 7. 响应式考虑

### 7.1 最小宽度
- 面板最小宽度: `320px`

### 7.2 最大宽度
- 消息最大宽度: `90%`

### 7.3 滚动行为
- Messages区域: `overflow-y: auto`
- 自动滚动到底部

## 8. 动画

| 动画 | 时长 | 缓动 |
|------|------|------|
| 消息出现 | `0.2s` | `ease-out` |
| 历史面板展开 | `0.2s` | `ease-in-out` |
| 按钮悬停 | `0.15s` | `ease` |
| 加载动画 | `1s` | `linear` (循环) |

## 9. 无障碍

- 所有交互元素支持键盘导航
- 颜色对比度符合WCAG AA标准
- 图标添加文字描述
- 状态变化有明确的视觉反馈
