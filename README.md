# AI Novel Writer

Phase 1 scope:

- Home page
- Book list
- Create book
- AI config
- AI test chat

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Env

Use the UI to save AI config, or prepare:

```env
AI_API_KEY=
AI_BASE_URL=https://api.openai.com/v1
AI_MODEL=gpt-4o-mini
```
